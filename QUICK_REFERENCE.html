    Common Tasks Quick Steps

    Update Main Menu

    1. Shopify Admin → Navigation → Edit menu
    2. Theme Customizer → Header → Select menu → Save

      3. Add Product to Site

      1. Products → Add product
      2. Fill in details, images, variants
      3. Set to Active → Save

        4. Change Product Price

        1. Products → Open product
        2. Variants section → Edit price → Save

          3. Update Footer Menu

          1. Shopify Admin → Navigation → Edit menu
          2. Theme Customizer → Footer → Select menu → Save

            3. Edit Page Content

            1. Online Store → Pages → Open page
            2. Edit content → Save

              3. Product Template Reference

              Template When to Use
              Default product template Standard products
              HEX® visualization template HEX® products with visualization
              MyoVolt product template MyoVolt products

              Menu Item Types

              • Collection - Links to product collection
              • Page - Links to content page
              • Product - Links to specific product
              • HTTP - Custom URL (internal or external)

              Footer Column Structure

              • Column 1: Logo, Newsletter, Social Icons
              • Column 2: Menu (configurable title)
              • Column 3: Menu (configurable title)
              • Column 4: Menu (configurable title)
              • Column 5: USB Logo, Payment Icons, Copyright

              Key Metafields

              • custom.best_for - Product use case
              • custom.product_family - Product line
              • custom.features - Product features (metaobject reference)
              • myovolt. - MyoVolt-specific data
              • pdp.swatch_color - Color swatch display

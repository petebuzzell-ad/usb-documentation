Documentation Methodology

This documentation follows the ARCDIG-DOCS methodology (v1.5.0) and uses an evidence-based approach:

  • All menu assignments verified against theme code
  • Metafield documentation based on actual data exports
  • Only apps with theme code integration are documented
  • Code references include file paths and line numbers

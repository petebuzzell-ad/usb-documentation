# Assets Directory

This directory contains visual assets required for the RUDIS documentation.

## Required Assets

### Logo Files
- **rudis-logo.png** - RUDIS company logo (for header)
  - Recommended size: 200px width minimum
  - Format: PNG with transparency
  - Location: `assets/rudis-logo.png`

### Icons
- **home-icon-svgrepo-com.svg** - Home icon for navigation
  - Format: SVG
  - Location: `assets/home-icon-svgrepo-com.svg`
  - Source: https://www.svgrepo.com/svg/13609/home

- **burger-menu-svgrepo-com.svg** - Hamburger menu icon
  - Format: SVG
  - Location: `assets/burger-menu-svgrepo-com.svg`
  - Source: https://www.svgrepo.com/svg/36160/hamburger-menu

### Favicon
- **favicon.webp** - Site favicon
  - Format: WebP
  - Size: 32x32px or 64x64px
  - Location: `assets/favicon.webp`

### Branding
- **Green_Arcadia Digital.png** - Arcadia Digital logo for footer
  - Format: PNG
  - Location: `assets/Green_Arcadia Digital.png`
  - Note: This should be the green Arcadia Digital logo

## CSS Styling

The documentation requires the `arcadia-style.css` file in the root directory. This file contains:
- Arcadia Digital brand colors (forest green palette)
- Typography (Piazzolla/Figtree fonts)
- Header and footer styling
- Two-column layout for documentation pages
- Enhanced TOC styling
- Mobile responsive design

## Getting Assets

1. **RUDIS Logo:** Obtain from RUDIS brand guidelines or website
2. **Icons:** Download from SVGRepo or use provided sources
3. **Arcadia Logo:** Use existing Arcadia Digital green logo
4. **Favicon:** Create from RUDIS logo or use brand icon

## Notes

- All image paths in HTML files reference `assets/` directory
- SVG files are preferred for icons (scalable, small file size)
- PNG files are used for logos (better support for complex images)
- Ensure all assets are optimized for web (compressed appropriately)
